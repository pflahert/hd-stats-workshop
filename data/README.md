# ALL Dataset

Acute Lymphoblastic Leukemia (ALL) gene expression data exported from the Bioconductor `ALL` package for use in Python-based workshop labs.

## Files

- **all_expression.csv** — 12,625 genes (rows) x 128 samples (columns). Log2-scale microarray expression values.
- **all_metadata.csv** — 128 rows with columns: `sample_id`, `BT` (original Bioconductor labels), `subtype` (simplified: B or T).

## Provenance

- **Source**: Bioconductor `ALL` package (version 1.44.0)
- **Original study**: Chiaretti et al. (2004). Gene expression profile of adult T-cell acute lymphocytic leukemia identifies distinct subsets of patients with different response to therapy and survival. *Blood*, 103(7), 2771-2778.
- **Platform**: Affymetrix HGU95Av2 microarray
- **Samples**: 128 patients (95 B-cell ALL, 33 T-cell ALL)

## Export

Generated by `export_all.R` using the Bioconductor ALL package in R.

## Usage (Python)

```python
import pandas as pd

url_base = "https://raw.githubusercontent.com/pflahert/hd-stats-workshop/main/data/"
expr = pd.read_csv(url_base + "all_expression.csv", index_col=0)
meta = pd.read_csv(url_base + "all_metadata.csv")
```

High-Dimensional Data Analysis Workshop

Biotechnology Training Program

Published

May 1, 2026

Workshop Overview

This two-day workshop introduces graduate students in the Biotechnology Training Program to the statistical challenges and modern methods for analyzing high-dimensional data—the setting where the number of measured variables \(p\) far exceeds the number of observations \(n\). This regime is now routine in genomics, proteomics, imaging, and many other areas of biotechnology.

References

  • Efron, B. (2010). Large-Scale Inference: Empirical Bayes Methods for Estimation, Testing, and Prediction. Cambridge University Press.
  • Hastie, T., Tibshirani, R., & Friedman, J. (2009). The Elements of Statistical Learning (2nd ed.). Springer.
  • Hastie, T., Tibshirani, R., & Wainwright, M. (2015). Statistical Learning with Sparsity. CRC Press.
  • James, G., Witten, D., Hastie, T., & Tibshirani, R. (2021). An Introduction to Statistical Learning (2nd ed.). Springer.

Learning Outcomes

By the end of this workshop, students will be able to:

# Learning Outcome Key Assessments
LO1 Explain why classical statistical inference breaks down when \(p \gg n\) and apply false discovery rate (FDR) controlling procedures to large-scale testing problems. Pre-HW, Lab 1, Lab 2, HW Q1–Q2
LO2 Apply dimension reduction and regularized regression methods to extract meaningful structure from high-dimensional biological datasets. Lecture 3 discussion, Lab 3, Lecture 4 discussion, Lab 4
LO3 Critically evaluate computational analysis output—including catching errors in code, questioning default choices, and verifying biological interpretations against primary sources. All Labs, Pre-HW, HW Q3

Software Requirements

Students should have the following installed before Day 1:

  • R (≥ 4.3) with packages: tidyverse, limma, qvalue, glmnet, ggplot2, pheatmap, Rtsne, uwot
  • Python (≥ 3.10) with packages: numpy, scipy, pandas, scikit-learn, statsmodels, matplotlib, seaborn, umap-learn
  • Quarto (≥ 1.4) for rendering notebooks
  • University AI platform (or Claude / ChatGPT) — used in every lab session

Reference Materials

  • Decision Guide for High-Dimensional Analysis — A one-page reference with flowcharts for choosing methods (error rates, dimension reduction, penalized regression) and a typical analysis workflow. Use this during the labs and take it with you.

Grading Philosophy

This is a training workshop, not a graded course. Assessments are formative—they exist to consolidate understanding and give you feedback. Engage seriously with them; the value is in the doing.

AI Use

You may use generative AI tools in the labs and homework. Some exercises will ask you to do so explicitly. The assessment focuses on whether you understand what the analysis does, whether it’s correct, and what the results mean.
